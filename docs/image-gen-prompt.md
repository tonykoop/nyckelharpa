The signature recognizable feature is the keybox carrying rows of slender sliding wooden keys, each key terminating in a small wooden tangent oriented to stop a bowed melody string from the side. ## Negative / Avoid

No text, watermarks, logos, or fake maker marks; no dimension lines or measurement callouts; no impossible string paths or mechanisms that block bow access or trap the sympathetic strings; no decorative-only keys; sympathetic strings never appear bowed or stopped by keys; anatomically plausible hands if a player is shown.
