# Tolerance And Fit Analysis

Status: L3-candidate open-question analysis. This file names critical fits and
clearances, but it does not assign fabricated values. Every clearance,
dimension, force, and tuning-dependent relationship remains
`estimate_pending_measurement`.

## Open Fit Questions

| interface | open question | why it matters | evidence needed before release |
| --- | --- | --- | --- |
| key to side guide | What side clearance lets the key slide after finish and humidity change without audible rattle? | Too tight causes binding; too loose moves the tangent contact point. | Bench keybox trials with repeated travel checks and side-play observations. |
| key to top and bottom guide | What vertical clearance prevents lift, scrape, and bounce during return? | Vertical wander changes tangent approach and release behavior. | Travel tests under representative key force and return method. |
| key end stop to keybox | How should travel be limited without hammering, noise, or tangent over-contact? | End-stop behavior sets tactile feel and protects string/tangent contact. | Stop material and geometry trials with noise notes. |
| tangent stem to key | How much adjustment range is needed before the tangent joint becomes fragile? | The tangent must be set after real string paths are known. | One-key rig with repeated adjustment and retention checks. |
| tangent face to melody string | What contact face shape stops cleanly without scraping or crushing? | This is the pitch-setting and tone-setting contact. | Stopped-string trials logged for clean speech, buzz, drag, and release. |
| released tangent to vibrating melody string | What release clearance survives key return variation and string motion? | Insufficient clearance causes ghost contact and buzz. | Open rig with visible release checks under bowed excitation. |
| melody string to adjacent melody string | What spacing permits bowing and tangent access for the chosen reference style? | Spacing affects bridge crown, bow access, and keyboard layout. | Measured reference or playable mockup, checked at nut, tangent zone, and bridge. |
| melody string to keybox wall | How much lateral room is needed for vibration, tangent approach, and service? | Crowding can cause buzz and makes setup impossible. | String-path mockup with service-tool access observations. |
| sympathetic string to key underside | What clearance prevents accidental contact while keeping the resonance path close enough to couple? | Contact creates noise; excessive separation may reduce intended response. | Transparent or open-keybox mockup with moving-key checks. |
| sympathetic string to tangent | Where can sympathetic paths pass without tangent collision during press and release? | Moving tangents are the highest-risk interference source. | Full travel sweep with all tangent families represented. |
| sympathetic string to service opening | How can strings be replaced and tuned without removing the key action datum? | Service access is part of reliability, not a cosmetic feature. | Maintenance rehearsal on mockup or prototype. |
| bridge feet to soundboard | What foot fit and bearing area support bowed-string load without choking response? | Bridge fit affects tone and structure. | Chalk-fit or equivalent contact record from prototype, with no tonal authority until tested. |
| bridge crown to bow path | What crown profile gives access to each bowed string for the chosen string layout? | Crown sets playability and adjacent-string interference. | Bowing trial across the actual string layout. |
| nut slots to string gauges | Which slot shapes match the eventual strings without pinch, buzz, or sideways wander? | String gauge is not selected yet, so nut work cannot be final. | Slot tests after string family selection. |
| tailpiece or anchor to body | How is load introduced without distorting body or moving bridge alignment? | Anchor movement changes every string and tangent relationship. | Structural mockup and recheck of bridge/string datums. |
| keybox to body mount | Can the keybox be removed and reinstalled without losing the tangent-string relationship? | Serviceable design requires repeatable datums. | Removal and reinstallation trials with post-checks. |
| removable cover to key clearance | How much cover clearance prevents contact while protecting the action? | Covers must not become hidden moving-part interference. | Closed-cover travel test after finish allowance is considered. |
| finish buildup to sliding fits | Which surfaces must be masked, burnished, waxed, or left unfinished? | Finish can consume clearances and create seasonal binding. | Finish coupon and guide-surface trial. |

## Tolerance Stack Questions

- Does the body centerline remain the master datum, or should the melody string
  plane become the master datum after setup?
- Which fits are allowed to move during setup, and which must be fixed before
  tangent positioning begins?
- Should tangent height adjustment absorb body/keybox variance, or should the
  keybox mount include shimming authority?
- Which service operations are allowed to disturb the keybox datum?
- How will seasonal movement be separated from build error during validation?

## Release Boundary

These questions are intentionally unresolved. The next packet may answer them
only with measured references, prototype logs, or reviewed design-table rows.
Until then, the tolerances remain open and the register authority remains
`concept_only` or `pending_measurement`.
